Sorting Visualizer

Sorting Visualizer is a web-based tool that allows users to visualize sorting algorithms.

Features

Supports multiple sorting algorithms:

Bubble Sort

Selection Sort

Insertion Sort

Merge Sort

Quick Sort


Technologies Used

Frontend: TypeScript, React (Next.js)

Libraries: Framer Motion, Tailwind CSS
